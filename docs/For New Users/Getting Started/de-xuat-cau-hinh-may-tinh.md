Xin chào, khi sử dụng phần mềm thiết kế 3vjia, bạn nên sử dụng Google Chrome hoặc ứng dụng client của 3vjia.

Hệ điều hành: Hỗ trợ Windows 7, Windows 8, Windows 10, không hỗ trợ hệ điều hành XP.
