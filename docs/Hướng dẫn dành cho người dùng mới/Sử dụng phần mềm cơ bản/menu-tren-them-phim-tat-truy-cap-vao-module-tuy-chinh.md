II. Các bước thao tác:

1. Mở 3D và vào module tùy chỉnh;

2. Di chuột qua biểu tượng **Quay lại (Back)** và nhấp để quay lại module cơ bản;

3. Di chuột qua tiêu đề của module tùy chỉnh để mở rộng menu tùy chỉnh.
